  <h3 class="relative mt-8 text-xs pl-4">
    <span class="absolute left-0 top-0 text-blue-500 font-black">1</span>
    Project X.E.N.O. was originally designed to work exclusively in TTT gamemode. Based on user feedback, it was later expanded to also function in Sandbox mode.
  </h3>
  <h3 class="relative mt-2 text-xs pl-4">
    <span class="absolute left-0 top-0 text-blue-500 font-black">2</span>
    X.E.N.O. is built from <a class="text-blue-500 hover:text-blue-600 hover:underline decoration-2 duration-500 font-bold" target="_blank" rel="noopener noreferrer" href="https://www.half-life.com/en/halflife2">Half Life 2</a> models in Garry’s Mod. Its main body is a single model that gives it an alien-like look, while on its stomach two additional models are combined to form an attachment that generates grenades. One part of this attachment is a bomb-like element that rotates continuously for a cool effect.
  </h3>
  <h3 class="relative mt-2 text-xs pl-4">
    <span class="absolute left-0 top-0 text-blue-500 font-black">3</span>
    All sound effects used in Project X.E.N.O. were generated using AI, with <a class="text-blue-500 hover:text-blue-600 hover:underline decoration-2 duration-500 font-bold" target="_blank" rel="noopener noreferrer" href="https://loudme.ai/">LoudMe</a> as the tool for sound creation.
  </h3>
  <h3 class="relative mt-2 text-xs pl-4">
    <span class="absolute left-0 top-0 text-blue-500 font-black">4</span>
